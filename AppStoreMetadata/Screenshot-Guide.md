## üé® Using screenshots.pro

### Step 1: Prepare Your Screenshots

**Before using screenshots.pro:**
1. Take raw screenshots on your iPhone (no device frame)
2. Use **iPhone 14 Pro Max or later** for native 1290x2796 resolution
3. Take screenshots in both **Dark Mode and Light Mode**
4. Choose the best-looking mode for each screen (likely Dark Mode for workout execution)

**Screenshot Checklist:**
- ‚úÖ No status bar clutter (9:41 AM is Apple's standard time)
- ‚úÖ Full battery indicator
- ‚úÖ Strong cellular signal (or Wi-Fi)
- ‚úÖ No notification badges on tab bar
- ‚úÖ No debug data visible
- ‚úÖ Professional sample data (no "Test 123" names)

### Step 2: Upload to screenshots.pro

1. Go to **https://screenshots.pro**
2. Click "Create Screenshots"
3. Upload your raw screenshots (PNG or JPG)
4. Select device frame options:
   - **Device:** iPhone 15 Pro Max
   - **Color:** Space Black, Natural Titanium, or White Titanium (choose based on app theme)
   - **Orientation:** Portrait
   - **Background:** Gradient or Solid (choose to match app branding)

**Background Suggestions:**
- For FittyPal: Use gradient matching app theme
  - Dark mode screenshots: Purple/blue gradient (matches app)
  - Light mode screenshots: Orange/peach gradient
- Alternatively: Clean white or black background

### Step 3: Add Text Overlays (Optional)

**Apple's Guidelines:**
- Text overlays are allowed but not required
- Keep text minimal and readable
- Use clear, sans-serif fonts (SF Pro recommended)
- Don't obstruct UI elements

**Text Overlay Ideas:**
1. "Real-time Heart Rate Zones" (Screenshot #1)
2. "Unlimited Workout Programs" (Screenshot #2)
3. "50+ Exercises Included" (Screenshot #3)
4. "Personalized Training Zones" (Screenshot #4)
5. "Track Every Workout" (Screenshot #5)

**Placement:**
- Top: Above device frame
- Bottom: Below device frame
- Avoid covering app UI

### Step 4: Export Screenshots

**Export Settings:**
1. Format: **PNG** (best quality, required by Apple)
2. Resolution: Original (1290x2796 for iPhone 6.7")
3. Quality: Maximum
4. Include device frame: Yes
5. Include background: Yes (if you added one)

**Download all variations:**
- iPhone 6.7" (1290 x 2796) - 5 screenshots
- iPhone 6.5" (1242 x 2688) - 5 screenshots
- iPad Pro 12.9" (2048 x 2732) - 5 screenshots

## üé® Design Tips for Maximum Impact

### Use Real Data
- Real exercise names: "Barbell Bench Press", "Romanian Deadlift"
- Realistic weights: 60kg, 80kg, 100kg (not 999kg)
- Realistic reps: 8-12 (not 100 reps)
- Professional client names: "Marco R.", "Sarah K." (not "Test Client")

### Highlight Unique Features
- Heart rate zones with colors (your competitive advantage)
- Multiple training methodologies (cluster sets, rest-pause)
- Clean, modern UI design
- Privacy-first messaging (if adding text overlays)

### Consistency
- Use same device frame color for all screenshots
- Use same background style (gradient or solid)
- Maintain same text overlay style if used
- Keep same mode (all dark or all light, don't mix)

### Visual Appeal
- Choose screens with vibrant colors (HR zones are colorful)
- Show variety (different exercises, different workouts)
- Avoid busy/cluttered screens
- Use screens with visual hierarchy

## üí° Pro Tips

1. **First Screenshot is Critical** - It's the only one guaranteed to be seen in search results. Make it your best.

2. **Show the App in Action** - Screenshots of active workouts are more engaging than empty states.

3. **Use Dark Mode** - Fitness apps often look better in dark mode (more dramatic, focuses attention).

4. **Avoid Clutter** - Don't show too many features in one screenshot. One feature per image.

5. **Test on Device** - Before finalizing, view screenshots on an actual iPhone to ensure readability.

6. **A/B Test** - After launch, you can update screenshots. Monitor conversion rates and iterate.

7. **Video Preview** - Consider adding an App Preview video (15-30 seconds) showing workout execution flow. This can significantly boost conversions.
